## 📥 Installation

### Step 1: Download
Download the latest release from the [Releases](https://github.com/EliteCDA/Illusion-Client/releases) page.

### Step 2: Install
1. Extract the downloaded `.zip` file to a folder of your choice
2. Run the installer or launcher included in the package
3. Select your Minecraft installation path
4. Click "Install"

### Step 3: Launch
Launch the client from your applications or use the provided launcher shortcut.

### Step 4: First Launch
The client will create a configuration folder on first launch. You can now configure settings.

> **Tip:** On first launch, try pressing **RIGHT SHIFT** to see the settings menu. Your UI animations are enabled by default—disable them in competitive modes if you need every FPS.

## ⚙️ Configuration

### Settings Location
Configuration files are stored in your Minecraft folder under `.minecraft/illusion/`

### Basic Setup
1. Open the client and press **RIGHT SHIFT** (or hit **ESC** in the main menu)
2. Navigate to **Performance** to adjust FPS vs. visuals balance
3. Go to **PvP Settings** to tune combat controls
4. Toggle **UI Animations** if you want full performance mode
5. Save your profile

### Common Configurations

**Maximum FPS (Competitive)**
- Reduce render distance to 6-8 chunks
- Disable fancy graphics
- Set particles to "Minimal"
- Lower brightness/contrast
- **Disable UI Animations** (RIGHT SHIFT)

**Balanced**
- Render distance: 12 chunks
- Enable fancy graphics
- Moderate particle effects
- Standard brightness
- UI Animations enabled

**Maximum Quality**
- Render distance: 16+ chunks
- Enable all visual enhancements
- Full particle effects
- Custom shaders enabled
- UI Animations fully enabled

## 🔧 Troubleshooting

**Client won't launch?**
- Ensure Java 17+ is installed: `java -version`
- Try deleting `.minecraft/illusion/` and relaunching

**FPS not improving?**
- Make sure you're on the "Maximum FPS (Competitive)" preset
- Check that you've disabled fancy graphics in Settings �� Performance
- Verify your render distance is 8 chunks or lower

**Animations causing lag in competitive play?**
- Press **RIGHT SHIFT** to toggle animations OFF
- Disable all visual effects in Settings → Graphics

**Game crashes on startup?**
- Update to Java 21 (recommended)
- Check that Minecraft 1.21.11 is installed
- Clear your launcher cache

**Still stuck?** Check the [Issues](https://github.com/EliteCDA/Illusion-Client/issues) page or create a new issue with your system specs.
